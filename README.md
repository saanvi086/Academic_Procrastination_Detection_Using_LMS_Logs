Academic Procrastination Detection Using LMS Logs

Minor Project | BTP2CSE044 | IILM University, Greater Noida | Session 2025–26

📌 Project Overview
This project aims to detect academic procrastination behavior in students using Learning Management System (LMS) activity logs and Machine Learning techniques. By analyzing patterns in login frequency, assignment submission timing, and resource access, the system identifies at-risk students early so timely interventions can be made.
